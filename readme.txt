MiniLog is a minimal WordPress theme most suitable for bloggers who would like to go with a responsive, simple, no-nonsense interface.

== Description ==
MiniLog is a minimal, responsive theme that comes built-in with features like featured-images, breadcrumbs, gallery etc and is translation ready.
